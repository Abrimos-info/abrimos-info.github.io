Conocimiento estratégico a partir de datos


  • Creamos infraestructura digital con información fundamental para las democracias de América Latina.

  • Acompañamos a organizaciones, gobiernos y empresas para que conviertan datos e información en insumos para la toma de decisiones estratégicas.

  • Generamos narrativas basadas en evidencia que empoderan comunidades, impactan a actores relevantes y promueven la justicia social.

Líneas de trabajo

En Abrimos.info combinamos nuestra experiencia abriendo datos con amplias capacidades narrativas, investigativas y de desarrollo para crear bases de datos fiables y verificadas que promueven el conocimiento estratégico.

Nuestro software base, TeseoETL, extrae, estructura, crea un sistema de análisis y publica datos en plataformas para hacer frente a la desinformación que afecta a las democracias de todo el mundo. Lo aplicamos a estas cuatro líneas de trabajo:

Promovemos el acceso a la información pública

pidala.info

Exploramos contrataciones públicas y beneficiarios finales

sociedad.info

Publicamos el histórico de datos electorales y de gobernantes

gobernantes.info

Aceleramos tu proyecto con desarrollo, investigación o formación
