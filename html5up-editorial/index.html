Conocimiento estratégico a patir de datos

Abrimos.info busca a generar conocimiento estratégico a partir de abrir y usar datos que son cruciales para la democracia y la rendición de cuentas. Este trabajo lo hacemos promoviendo la tecnología cívica, generando narrativas de cambio y con una perspectiva latinoamericana. Además ofrecemos servicios de ETL y herramientas de fácil uso para extraer conocimiento de todo tipo de datos.
